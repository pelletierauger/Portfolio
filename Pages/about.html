Ce site Web

Pour réaliser ce site Web, j'ai développé une application Node.js qui lit une base de données puis génère automatiquement tous les fichiers html. J'ai créé cette application parce que je cherchais un « générateur de site statique » (static site generator) à la fois simple et offrant quelques fonctionnalités qui me sont importantes (tel que la possibilité de gérer une base de données de contenu bilingue), et j'étais insatisfait des générateurs déjà existants. Le code source de cette application est sur GitHub (avec le code source de ce site Web au complet). Préparez-vous à
une absence complète
de documentation.Le module principal est maker.js. Il lit les données présentes dans les dossiers pages et blog, et utilise des modules supplémentaires contenus dans les dossiers formatters et node_modules. Les formules mathématiques affichées sur le site ont été mises en forme avec le module Mathjax-Node et la syntaxe des boîtes de code informatique a été colorée avec le module Highlight.js ainsi qu'avec un peu de code supplémentaire.

This website

To make this website, I developed a Node.js application that reads a database and outputs html content, generating all the files automatically. I created it because I wanted a lightweight static site generator with a few key features (like the possibility of easily managing a database of bilingual content), and I wasn't satisfied by pre-existing generators. The source code for this application is on GitHub (along with the source code for this whole website).Prepare for
a complete lack
of documentation. The main module is maker.js. It reads the data inside the pages and blog folders and uses additionnal modules found in the formatters and node_modules folders. The mathematical formulas displayed on this site are formatted with the Mathjax-Node module and the syntax of code boxes is coloured with the Highlight.js module and with some extra custom code.
